欢迎使用Simple Writer，它是一个简洁的写作工具
By 陈思全
为什么编写Simple Writer?
大家应该用过Word 文档吧，在那里写文章十分的不舒服。总是要切换各种各样的东西，写作的灵感完全没了。。。
其实写作并不需要很强大的工具，而是一个不易让人分心的环境。其实写作并不需要很强大的工具，而是一个不易让人分心的环境。
我就想了想，为什么不自己动手做一个更简洁的写作工具呢？\n
于是，我就趁着暑假空闲时间，利用我自己学到的前端知识，编写了Simple Writer。
Simple Writer的特点
- 全自动保存。\n
- 快捷键全覆盖，几乎所有功能都有对应的快捷键。\n
- 灵感模式，不放过任何一个在脑海中一闪而过的灵感。\n
- 亮/暗色和护眼模式，不让灵感局限于时间。\n
- 沉浸式状态栏+全屏模式，让你专注于写作。\n
- 支持另存为多种格式，TXT、MarkDown、DOC、HTML都不在话下。\n
- 完全本地保存，绝对隐私保护。\n
- 支持数据导入导出，换台设备也不怕。
Simple Writer 的诞生离不开：
- jQuery
       jQuery是一个流行的DOMJS库，它为Simple Writer提供了底层支持。
- Bootstrap Icons 和Font Awesome和IconMoon\n
      它们都为Simple Writer 提供了图标资源。
- Github Pages\n
      Github Pages是一个稳定的静态网页托管平台，Simple Writer的在线功能由他提供。
- Prism
      PrismJS提供代码高亮显示。
- Clear Writer 的少部分样式\n
      由于本人实在没有什么设计天赋，所以，少部分样式我参考了Clear Writer，它是一个很优秀的MarkDown写作工具（clearwriter.gitee.io）

Simple Writer 帮助
By 陈思全
快捷键使用帮助
基本快捷键
- Ctrl+C 复制
- Ctrl+X 剪切
- Ctrl+V 粘贴
- Ctrl+A 全选本编辑区域
- Ctrl+Z 撤回本编辑区域的文本
- Ctrl+Y 反撤回本编辑区域的文本
- Tab 向下切换编辑区域
- Shift+Tab 向上切换编辑区域
Editor快捷键
- Ctrl+Enter 在该小节下新建一个小节
- Ctrl+Alt+Enter 在该小小节或编辑区域下新建一个小小节
- Ctrl+Q 在该编辑区域下新建一个引用部分
- Ctrl+Alt+Q 在该小小节外新建一个引用部分
- Ctrl+P 在该编辑区域下新建一个文字域
- Ctrl+Alt+P 在小小节外新建一个文字域
- Ctrl+I 在该编辑区域下新建一个图片
- Ctrl+Alt+I 在小小节外新建一个图片
- Ctrl+D 在该编辑区域下新建一个代码域
- Ctrl+Alt+D 在小小节外新建一个代码域
- Delete 删除该编辑区域
- Ctrl+Delete 删除该小小节
- Ctrl+Shift+Delete 删除该小节
功能快捷键
- Ctrl+S 保存
- Ctrl+Alt+S 另存为
- Ctrl+Shift+S 打开设置
- F1 打开帮助
- F11 全屏
关于手机使用的问题
Simple Writer V2.0已停止对手机端的支持，如有不便，尽请谅解！
关于Editor的帮助
如何修改图片和代码域？
双击图片或代码域。
如何在图片或代码域下新建元素？
单击图片或代码域，然后执行相关操作。
为什么删除元素时，有的会有提示，有的不会有？
删除元素时，系统会根据内容的多少来判断是否提示。
每次新建文档，都会有一个默认的文字域，我不想要它，只要小节，怎么做？
你可以在文字域那里按Ctrl+Enter，然后确定你不需要它，再把它Delete删掉。反悔时，可以在副标题这里Ctrl+P。
关于文件的帮助
如何让灵感模式下的文件添加标题和副标题？
对不起，灵感模式是不能添加标题和副标题的。
另存为上面的格式都看不懂，该选哪个？
按需选择，推荐选择DOC（Word 文档）和HTML（既可以用浏览器打开，也可以用Word文档打开）
