欢迎使用Simple Writer，它是一个简洁的写作工具
By 陈思全
为什么编写Simple Writer?
大家应该用过Word 文档吧，在那里写文章十分的不舒服。总是要切换各种各样的东西，写作的灵感完全没了。。。
其实写作并不需要很强大的工具，而是一个不易让人分心的环境。
我就想了想，为什么不自己动手做一个更简洁的写作工具呢？
于是，我就趁着暑假空闲时间，利用我自己学到的前端知识，编写了Simple Writer。
Simple Writer的特点
- 全自动保存。
- 快捷键全覆盖，几乎所有功能都有对应的快捷键。
- 灵感模式，不放过任何一个在脑海中一闪而过的灵感。
- 亮/暗色模式，不让灵感局限于时间。
- 沉浸式状态栏+全屏模式，让你专注于写作。
- 支持另存为多种格式，TXT、MarkDown、DOC、HTML都不在话下。
- 完全本地保存，绝对隐私保护。
- 支持数据导入导出，换台设备也不怕。
使用技巧
- Simple Writer 正常情况下3分钟保存一次，你也可以`Ctrl+S`保存
- Ctrl+Alt+S另存为
- Ctrl+Shift+S打开设置
- Ctrl+H打开文件目录
- Ctrl+Alt+Enter和Ctrl+Enter向上下添加一个小节
- Ctrl+Alt+Q和Ctrl+Q向上下添加一个引用部分
- Delete 删除本编辑区域（慎按，无法撤回）
- Ctrl+Delete 删除本编辑区域所在的小节（慎按，无法撤回）
- 你也可以在右上角菜单上实现以上功能
Simple Writer 的诞生离不开：
- jQuery
- Bootstrap Icons 和Font Awesome和IconMoon
- Github Pages
- Clear Writer 的少部分样式
*注：由于本人实在没有什么设计天赋，所以，少部分样式我参考了Clear Writer，它是一个很优秀的MarkDown写作工具（clearwriter.gitee.io）
Simple Writer 帮助
By 陈思全
关于编辑器的帮助
Q:如何创建新的小节？
A:Ctrl+（Alt+）Enter或右上角菜单->编辑->添加一节到上/下方，详情请见关于我们->使用技巧
Q:如何创建新的引用部分？
A:Ctrl+（Alt+）Enter或右上角菜单->编辑->添加引用域到上/下方，详情请见关于我们->使用技巧
Q:如何切换输入域？
A:`Tab`或鼠标（手指）
Q:每次新建文档，都会有一个默认的文字域，我不想要它，只要小节
A:你可以在文字域那里按Ctrl+Enter，然后确定你不需要它，再把它Delete删掉（不可撤销）
Q:我不小心删了默认的文字域怎么办？
A:在副标题处Ctrl+Q向下新建一个引用部分，它会在后面自动跟一个文字域，然后再把新建的引用部分删掉。
如果你是在灵感模式下误删的，请重新开始!
关于文件的帮助
Q:如何让灵感模式下的文件添加标题和副标题？
A:对不起，灵感模式是不能添加标题和副标题的。
Q:另存为上面的格式都看不懂，该选哪个？
A:按需选择，推荐选择DOC（Word 文档）和HTML（既可以用浏览器打开，也可以用Word文档打开）
关于设置的帮助
Q:我可以自定义主题色吗？
A:抱歉，暂时不行。
Q:我是手机端访问的，为什么我切换了字体，结果都一样？
A:Simple Writer使用的是浏览器自带的CSS字体，因为字体文件会影响加载速度，而手机端自带的字体支持普遍较差。
